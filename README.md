# beyonce-app
This is a small app to get beyonce's albums from an api and render them. The tecnology used is Vue.js.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
